Vidyo Call
==========

This Jive app adds an app action to discussion, document and
blog post pages that allows user to initiate a video conference
call using the Vidyo system. It requires a custom Java service
that provides a REST/JSON api for Vidyo. 

The app uses the following third party libraries:
- JQuery UI http://jqueryui.com/
- Bootstrap http://twitter.github.io/bootstrap/
