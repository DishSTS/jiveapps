DL Share
========

This is a Jive app that adds an app action to discussion, document and
blog post pages that allows user to share the content within Jive with
people based on email distribution groups. 

Group information is retrieved via Jive Connects from a custom Java 
service that queries LDAP.

The app uses the following third party libraries:
- JQuery UI http://jqueryui.com/
- Bootstrap http://twitter.github.io/bootstrap/
